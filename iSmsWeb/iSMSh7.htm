Порядок действий по обновлению версии программных пакетов iSMS через удаление пакетов предыдущей версии

 

1.    Выгрузить конфигурацию через WEB-интерфейс iSMS

2.    Посмотреть, какие пакеты установлены:
dpkg –l | grep isms

3.    Остановить apache2, серверные модули:
service apache2 stop
ema stopww

4.    Удалить все установленные пакеты в следующем порядке:
dpkg -P isms
dpkg -P isms-utils
dpkg -P isms-ntp
dpkg -P isms-doc
dpkg -P isms-web-ssl
dpkg -P isms-web
dpkg -P isms-server
dpkg -P isms-db

5.    Удалить базу данных:
mysql -uroot -p -e "drop database if exists rsduadmin;" 
Проверить, что база удалена:
mysql -uroot -p -e "use rsduadmin;" 
В этом случае после ввода пароля появится сообщение об ошибке:
ERROR: Unknown database 'rsduadmin'

6.    Запустить установку пакетов в следующем порядке:
dpkg –i isms-db*.deb
dpkg -–force-all –i isms-web*.deb
dpkg –i isms-server*.deb
dpkg –i isms-web-ssl*.deb
dpkg –i isms-doc*.deb
dpkg –i isms-ntp*.deb
dpkg –i isms-utils*.deb

7.    Запустить серверные модули и apache2:
ema start
service apache2 start

8.    Загрузить сохраненную конфигурацию через WEB-интерфейс iSMS
