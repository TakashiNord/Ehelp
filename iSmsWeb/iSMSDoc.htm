Общие сведения

Назначение

Руководство описывает установку ПО Docker в ОС Linux.

ПО Docker предназначено для подготовки и запуска приложений внутри песочницы, называемой контейнером. Приложение со всеми зависимостями преобразуется в стандартизированный модуль. Модуль запускается внутри контейнера на любой операционной системе, которая поддерживает контейнеризацию Docker. Контейнер изолирует приложение от операционной системы. Приложение внутри контейнера получает однотипную среду и работает одинаково независимо от окружающей инфраструктуры.

Описана установка ПО Docker 18.06.3 на ОС Debian 8.5 для ПО «ПТК телемеханики iSMS» версий 1.19.1.х.

Необходимые знания

Для выполнения действий, описанных в руководстве, требуются следующие знания:

ü  Адрес компьютера для установки ПО Docker;

ü  Пароль пользователя root.

 

Установка приложения

Установка производится от имени пользователя root.

Порядок установки

1.  Установить библиотеки:

# dpkg -i libltdl7_2.4.6-2_amd64.deb

# dpkg -i libseccomp2_2.3.1-2.1+deb9u1_amd64.deb

2.  Установить ПО Docker:

# dpkg -i containerd.io_1.2.6-3_amd64.deb

# dpkg -i docker-ce_18.06.3~ce~3-0~debian_amd64.deb

3.  Проверить установку:

# docker --version

Docker version 18.06.3-ce, build d7080c1
