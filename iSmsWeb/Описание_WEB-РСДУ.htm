Список изменений:

WEB-РСДУ 1.19.1.0 (25.03.2019)
==============================
Новые функции:
* Редактор схем. Добавлен новый элемент "Подстанция" в палитру графических элементов с возможностью настройки отображения нескольких уровней напряжения; поддерживается импорт элемента из схем формата МОДУС.
* Редактор схем. Добавлен новый графический элемент "Направление перетока" в виде направленной стрелки; поддерживается импорт элемента из схем формата МОДУС.
* Редактор схем. Добавлен новый элемент "Вольтодобавочный трансформатор" в палитру графических элементов; поддерживается импорт элемента из схем формата МОДУС.
* Редактор схем. Добавлен новый элемент "Круг" в палитру графических элементов.
* Редактор схем. Добавлена возможность выбора дополнительных цветов "Класс напряжения" и "Прозрачный" (цвет фона) в качестве цвета состояний ВКЛЮЧЕНО/ОТКЛЮЧЕНО для выключателей. Поддерживается импорт данных цветов из схем формата МОДУС.

Улучшения:
* Редактор схем. Для элемента "Прямоугольник" добавлена возможность редактирования и сохранения свойства "Стиль линии" с вариантами "Сплошная" или "Пунктирная", добавлена возможность выбора "прозрачного" цвета фона и линии.

Исправлены ошибки:
* Просмотр/редактор схем. Исправлена ошибка изменения масштаба (комбобокс "Выбор масштаба").
* Редактор схем. Исправлено позиционирование текста при импорте из схем формата МОДУС.
* Редактор схем. Исправлена ошибка отображения рамки выделения (при работе со схемой в какой-то момент переставала появляться рамка выделения при выборе элемента).

WEB-РСДУ 1.18.1.2 (12.12.2018)
==============================
Исправлены ошибки:
* Просмотр панелей. Исправлено открытие панелей, хранящихся во вложенных папках. Уникальность имени файла поддерживается в рамках одной папки.

WEB-РСДУ 1.18.1.0 (28.09.2018)
==============================
Новые функции:
* Просмотр панелей. Добавлена поддержка графиков на WEB-панелях (элементы с типом STRIPCHART в CVI).
* Просмотр панелей. Добавлена поддержка отображения времени на WEB-панелях, аналогичное отображению времени на информационных панелях CVI (выводится либо локальное время, либо время сервера в зависимости от исходных настроек панели).

Улучшения:
* Редактор схем. Добавлен импорт из схемы формата МОДУС размера шрифта цифрового табло.
* Редактор схем. Добавлена обработка масштаба для элементов "Выключатель" и "Пересечение провода" при импорте из схемы МОДУС.
* Редактор схем. Добавлена обработка толщины линии для элемента "Шина" при импорте из схемы МОДУС.
* Редактор схем. Добавлена обработка начального состояния для элементов "Разъединитель-предохранитель" и "Лампа сигнальная" при импорте из схемы МОДУС.
* Редактор схем. Добавлена обработка ориентации для элементов "Конденсатор" и "Разъединитель напряжения" при импорте из схемы МОДУС.
* Виджеты. Добавлена возможность редактирования имени виджета (пункт контекстного меню "Переименовать" для компонента виджета).

Исправлены ошибки:
* Редактор схем. Исправлена обработка слоёв схемы при импорте из схем МОДУС (графические элементы могли перекрывать друг друга в неправильном порядке).
* Редактор схем. Исправлена ошибка импорта неподдерживаемого элемента "Направление перетока" из схем МОДУС (элемент "Направление перетока" раньше ошибочно заменялся на "Контакт дугогасительный")
* Редактор схем. Исправлен конвертер схем формата МОДУС: Скорректировано изображение дуги для элемента "Пересечение провода".
* Редактор схем. Добавлено свойство "Номер слоя" для "Реактор шунтирующий", "Заградительный фильтр", "ОПН без земли" (ранее отсутствовало).

WEB-РСДУ 1.17.1.0 (30.06.2017)
==============================
Новые функции:
* Просмотр панелей. Добавлено отображение опциональных значений параметров на WEB-панелях.
* Редактор схем. Добавлена поддержка новых символов "Разъединитель выдвижной", "Разрядник вентильный", "Контакт Дугогасительный", "Заградительный фильтр", "Двигатель постоянного тока", "Двигатель", "Синхронный двигатель", "Синхронный компенсатор", "Реактор шунтирующий", "ОПН", "Автотрансформатор 1-обмоточный", "Разрядник трубчатый", "Контейнер" и пр. изменения в отображении графических элементов редактора схем.

Улучшения:
* Просмотр панелей. Доработан конвертер панелей с поддержкой стилей табло, таким образом чтобы размеры оригинального табло ТИ совпадали с размерами в просмотрщике панелей iSMS.
* Просмотр панелей. Доработан конвертер панелей с поддержкой z-index, таким образом чтобы сохранялся порядок следования элементов, как это отрисовано в редакторе CVI.
* Просмотр панелей, просмотр схем. Оптимизировано обновление оперативных данных на панелях и схемах - сокращено количество обращений к серверной части при одновременном открытии нескольких панелей или схем.

Исправлены ошибки:
* Просмотр схем. Исправлено отображение состояния для элемента "Шасси".
* Просмотр панелей. Исправлена ошибка при настройке нескольких предпочтений на одну и ту же панель

WEB-РСДУ 1.16.3.0 (16.02.2017)
==============================
Новые функции:
* Добавлена конфигурация и отображение параметров из раздела "Диспетчерские графики".
* Добавлена возможность создания ярлыков на панели, которые в предпочтениях вложены в другие папки.
* Во всплывающей подсказке над элементом панели добавлено отображение имени параметра.
* Добавлена функция открытия WEB-ресурсов по нажатию на настроенную кнопку на информационной панели.
* Добавлена возможность работы WEB-сервера по протоколу https

Исправлены ошибки:
* Исправлено отображение единицы измерения во всплывающей подсказке к настроенному на источник значений элементу панели. Единица измерения могла быть перепутана в случае наличия пересекающихся идентификаторов в подсистеме сбора и технологической информации.
* Исправлено отображение значений на панелях: в случае присутствия на панели элементов настроенных на параметры из разных разделов (например, из технологической информации и сбора) данные отображались только для одной из подсистем.
