Обновление MySQL до версии 5.7.24 (требуется, чтобы обновить старые версии iSMS (до 1.18.2.0) пакетами).

Перед выполнением обновления обязательно сделать резервную копию конфигурации (выгрузить текущую конфигурацию)!

Порядок обновления:

1. Скопировать архив на удаленную isms по SCP:
# scp debian8debs.tar.gz ismsadmin@remote-isms:~

2. Зайти на isms и стать суперпользователем, пароль passme:
# ssh ismsadmin@remote-isms
ismsadmin@isms# su

3. Извлечь архив
# tar -xvf debian8debs.tar.gz

4. Остановить модули iSMS, и проверить что все сервисы not running.
  Если запущены, то еще раз ema stopww
# ema stopww && ema status
   ...
   elregd                                                      Not running
   pwswtd                                                      Not running
   sysmond                                                     Not running
   ==================================
   The state is UNKNOWN!
   ==================================

5. Начать устанавливать новые версии, завершится с ошибкой. Будет ругаться. Много конфликтов. Так надо, на шаге 7 они разрешатся.
# dpkg -i debian8debs/*.deb

6. Удалить старые версии mysql-server
# dpkg -r mysql-client-5.5 mysql-server-5.5 mysql-server-core-5.5

7. Установить новые версии.
   Во время установки предупредит о необходимости бекапа - нажать Ok и
   попросит ввести новый пароль для root от базы данных - ввести passme 2 раза.
   После этого шага все конфликты разрешатся.
# dpkg -i debian8debs/*.deb

8. Запустить скрипт обновления базы (пароль passme).
# mysql_upgrade -u root -p --force
   В конце вывода должны быть строки:
   ...
   Upgrade process completed successfully.
   Checking if update is needed.

9. Проверить версию mysql, должна быть 5.7.24. Если версия не совпадает, звони ЕМА
# mysql --version

10. Перезапустить mysql
# service mysql restart

11. Запустить серверные модули и проверить, что они все запустились (Running)
# ema start
# ema status
  ...
  arcwrite                                                    Running
  dcsd                                                        Running
  elregd                                                      Running
  pwswtd                                                      Running
  sysmond                                                     Running
  ==================================
  The state is MASTER!
  ==================================
