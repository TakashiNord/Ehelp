Подключение репозитория iSMS

1.   Прописать репозиторий iSMS в менеджере пакетов apt. Для этого в каталоге /etc/apt/sources.list.d создать файл ismsrepo.list с содержимым:

deb http://ismsrepo.orpr.ema/ stable main

где

ismsrepo.orpr.ema — имя машины в сети; это имя должно нормально разрешаться в IP адрес на iSMS, где производится настройка.

Примечание. Если у вас имеется готовый файл для репозитория, то его можно скачать с помощью команды wget

Например,

# wget -O /etc/apt/sources.list.d/ismsrepo.list http://ismsrepo.orpr.ema/ismsrepo.list

 

2.      Если для iSMS не указан DNS сервер или у указанных DNS серверов нет возможности разрешить это имя в IP, то есть 2 варианта решения.

a) указать IP адрес в файле ismsrepo.list

Например,

deb http://192.168.20.185/ stable main

 

b) прописать локальное разрешение имени в IP адрес; для этого надо добавить соответствующую строку в файл /etc/hosts.

Например,

192.168.20.185 ismsrepo.orpr.ema

 

Или можно это выполнить коммандой:

# echo "192.168.20.185 ismsrepo.orpr.ema" >> /etc/hosts

 

3.      Добавить ключ, которым подписан репозиторий iSMS, в доверенные ключи. Ключ можно получить по ссылке: repo_ema.pubkey.asc

Это можно сделать выполнив на iSMS команды:

# wget -O /root/repo_ema.pubkey.asc http://ismsrepo.orpr.ema/repo_ema.pubkey.asc

# apt-key add /root/repo_ema.pubkey.asc

 

4.   В завершение, необходимо обновить репозитории:

# apt-get update
